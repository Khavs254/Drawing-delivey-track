When on, plate marks (PLT/FL and PL/FL/FLT profiles) are grouped into one row per thickness on Excel import, with every mark kept in the expandable breakdown. Turn off to import each plate as its own item.
AI Model — Drawing & Delivery Note Reading
Used for all three AI uploads: GA / Part Drawing extraction, Workshop Issue drawing extraction, and Delivery Note OCR. Note: the GitHub Models and official DeepSeek options are text-only — they only work for the AI Clash Check; the three upload features will show an error asking you to pick a different provider.
Stored only in browser localStorage — sent directly to api.anthropic.com. Get a key at console.anthropic.com.
Stored only in browser localStorage — sent directly to generativelanguage.googleapis.com. Get a key at aistudio.google.com. Check your live quota/usage at aistudio.google.com/usage.
Stored only in browser localStorage — sent directly to api.moonshot.ai. Moonshot's API is meant for server-side use, so some networks/ad-blockers may block a direct browser call (CORS) — if uploads fail immediately with a network error on Kimi, that's why. Get a key at platform.moonshot.ai.
Stored only in browser localStorage — sent directly to api.cerebras.ai. Vision uploads use Gemma 4 31B, currently a public-preview multimodal model on Cerebras — accuracy and availability may be less stable than the other providers, and free-tier accounts have fairly tight rate limits. Get a key at cloud.cerebras.ai.
Stored only in browser localStorage — sent directly to models.github.ai. One token powers both the DeepSeek and Llama options above. Needs a fine-grained personal access token with "Models: read" permission. Text-only — used for the AI Clash Check, not image/PDF uploads. Get a token at github.com/settings/tokens. Heads up: GitHub Models is scheduled for full retirement on 30 July 2026, so treat this as short-term.
Stored only in browser localStorage — sent directly to api.deepseek.com using the deepseek-v4-flash model. Text-only for now — image/PDF support in DeepSeek's API is unconfirmed as of mid-2026, so this stays blocked for uploads until that's verified; used for the AI Clash Check. Not affected by the GitHub Models retirement. Get a key at platform.deepseek.com/api_keys.
